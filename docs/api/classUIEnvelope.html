Detailed Description

Envelope widget which contains exactly one other widget and changes its representation.

The envelope can do the following things:

  • If the enclosed widget does not spawn the whole display area it is assigned, the envelope can place it with a certain alignment.
  • The envelope can add a border to the inner widget. This border will be empty.
  • If the inner widget prefers to be smaller than it could be, the envelope can report a larger area to the caller.
